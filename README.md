## What's Here

### Board of Directors (`/bod`)

**[Build Your Own Board of Directors](https://dataalchemie.github.io/wendy-builds/bod/build-your-own-bod.html)**
A guided 5-stage Claude Code command that ends with a fully personalized advisory council of 8 real people — dead or alive — selected for your specific goals and blind spots. The output is a deployment-ready `/journey-bod` skill file.

**[Personal OS Intake](https://dataalchemie.github.io/wendy-builds/bod/personal-os-intake.html)**
The intake tool that powers the board-building process. Maps your operating domains, values, and growth edge before any advisor selection happens.

**[build-bod.md](bod/build-bod.md)**
The raw Claude Code command file. Open it to see how the skill is structured — stages, gates, orchestrator rules, synthesis protocol.

### AI Upskilling

**[AI Upskilling Roadmap](https://dataalchemie.github.io/wendy-builds/upskilling/ai-upskilling-roadmap.html)**
A structured roadmap handout from Data Alchemie Group for executives and teams building AI literacy. Covers foundation, application, and governance tiers.

### Personal Operating Rhythm

**[Personal Operating Rhythm](https://dataalchemie.github.io/wendy-builds/helm/build-your-own-helm.html)**
A personal goals and weekly rhythm system built from five Claude Code slash commands: plan the week, review the week, take a monthly pulse check, score the quarter, and draft the next one. Works in Claude (web, CLI, desktop), Codex, and Gemini. The longer you run it, the sharper it gets.

The five commands: `/weekly-checkin`, `/weekly-checkout`, `/pulse-check`, `/score-okrs`, `/draft-okrs`
